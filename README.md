## Containerization with Docker
- **Image**: 실행 환경의 템플릿 (읽기 전용)
- **Container**: 이미지로부터 생성된 실행 인스턴스 (독립 실행 환경)

### 이미지 커밋 & Dockerfile
- 컨테이너 상태를 저장: `docker commit`
- 설정 기반 자동 이미지 빌드: `Dockerfile`
